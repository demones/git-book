# git-book

基于 git 官方 https://www.git-scm.com/book/zh/v2 生成在线 [html git book](http://demones.github.io/git-book/)

方便离线学习 git

当前版本 progit-zh.936

直接从官方 copy，没有修改官方文档，之所以要在 github 上发一份，是因为方便手机端阅读，
官方的文档在手机端阅读不是很方便。
